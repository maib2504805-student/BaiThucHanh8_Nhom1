1. Giới thiệu

Bài Lab 08 nhằm mục đích thực hành cấu hình mạng cơ bản bằng Router và Switch.
Sinh viên sẽ thực hiện các bước tạo bảng địa chỉ IP, cấu hình thiết bị mạng và kiểm tra kết nối giữa các máy tính trong mạng.

2. Nội dung thực hành
Bước 1: Kẻ bảng địa chỉ IP

Tạo bảng gồm các thông tin:

Host

IP Address

Gateway

DNS Server

Bước 2: Trình bày bảng Gateway, DNS server và DHCP
Gateway
DNS Server
DHCP Server
Bước 3: Trình bày các loại địa chỉ IP

Liệt kê các loại địa chỉ IP như:

Địa chỉ IP tĩnh (Static IP)

Địa chỉ IP động (Dynamic IP)

Địa chỉ IP Public

Địa chỉ IP Private

Bước 4: Trình bày chức năng của Router và Switch

Router

Kết nối các mạng khác nhau

Định tuyến dữ liệu giữa các mạng

Cấp phát IP nếu cấu hình DHCP

Switch

Kết nối các thiết bị trong cùng mạng LAN

Chuyển tiếp dữ liệu dựa trên địa chỉ MAC

Giúp mở rộng mạng nội bộ

Bước 5: Trình bày kết quả kiểm tra kết nối

Kiểm tra kết nối bằng lệnh ping giữa các máy tính.

Ví dụ:

ping 192.168.1.1
ping 192.168.1.2

Nếu nhận được phản hồi Reply from... nghĩa là kết nối thành công.

3. Tìm hiểu thông tin địa chỉ IP

Thực hiện các bước sau:

Tìm hiểu thông tin địa chỉ IP của máy tính đang sử dụng.

Kiểm tra bằng lệnh:

ipconfig

Ghi lại các thông tin:

IPv4 Address

Subnet Mask

Default Gateway

DNS Server

4. Sử dụng lệnh Tracert

Lệnh tracert dùng để kiểm tra đường đi của gói tin từ máy tính đến server đích.

Ví dụ:

tracert www.google.com

Kết quả cho biết các bước trung gian mà gói tin đi qua trước khi tới server.

5. Kết luận

Qua bài Lab này, sinh viên:

Hiểu được cách cấu hình địa chỉ IP.

Biết chức năng của Router và Switch.

Sử dụng các lệnh kiểm tra mạng như ping, ipconfig, tracert.

Kiểm tra và đánh giá kết nối mạng trong hệ thống.
